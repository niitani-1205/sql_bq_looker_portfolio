## 📁 01_data_mart / SQL データマート

このフォルダには、StackOverflow 公開データを用いた
分析用 SQL を格納しています。

SQL は BigQuery 上で実行し、年度ごとの傾向を分析するために
質問データと回答データを結合し、以下の指標を生成しています：

- 最初の回答が付くまでの時間（分）
- 1時間以内 / 24時間以内の回答率
- 未回答フラグ
- 1年間未回答だった割合
- 年度別の質問数・回答数

---

### 📄 含まれている SQL

#### 1. 年度別_質問数と回答数.sql  
- 質問数・回答数を年度ごとに集計  
- コミュニティの活動量の推移を把握

#### 2. 年度別_質問回答速度.sql  
- 最初の回答までの時間を計算  
- 1時間以内 / 24時間以内の回答率を算出

#### 3. 年度別_1年経過未回答率.sql  
- 最初の回答が1年以上つかなかった質問を抽出  
- StackOverflow の未解決率の変化を可視化

---

### 🛠 使用技術

- BigQuery (Standard SQL)
- 公開データ：`bigquery-public-data.stackoverflow`  
  - posts_questions  
  - posts_answers

---

Looker Studio によるダッシュボード可視化のため、  
分析に適した形へ前処理・加工したデータマートを作成しています。
